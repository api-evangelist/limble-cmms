# Limble CMMS (limble-cmms)

Limble CMMS is a cloud-based computerized maintenance management system (CMMS) for manufacturing, facilities, and fleet maintenance teams. The Limble REST API V2 provides programmatic access to assets, locations, work orders, preventive maintenance schedules, parts inventory, purchase orders, vendors, users, and webhooks across regionally distributed endpoints covering North America, Canada, Australia, and Europe.

## APIs

### Limble API V2

The Limble CMMS REST API V2 provides HTTPS access to maintenance management data including assets, locations, work orders, preventive maintenance tasks, parts, purchase orders, users, vendors, teams, roles, budgets, and webhooks. Uses Basic authentication. Regional endpoints available for US, Canada, Australia, and Europe, plus a dedicated 21 CFR Part 11 compliant endpoint for pharmaceutical and life sciences manufacturing.

- **Human URL:** [https://apidocs.limblecmms.com/](https://apidocs.limblecmms.com/)
